# WordPress Theme Import Guide

Import the files in this order:

## 1. Theme

Go to **Appearance > Themes > Add New > Upload Theme**.

Upload `wordpress-starter.zip`, install it, and activate it.

## 2. ACF

Go to **ACF > Tools > Import Field Groups**.

Import `acf-export-2026-06-15.json`.

## 3. WordPress Content

Go to **Tools > Import > WordPress**.

Import `testing-twig.WordPress.2026-06-15.xml`.
